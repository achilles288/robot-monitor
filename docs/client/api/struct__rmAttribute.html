Detailed Description

Attributes shown by the client or manipulated by the station.

A key value pair structure that can be used in a map structure. This type of data is designed to be portable in communication between two devices. The data here is the real-time data of the client device which can also be overriden by the station. The value may be a string, a number or a blob.
